# How to use

## dev

build and host code on `http://localhost:1234`.  
build code output to `dist` directory.

```
npm run dev
```

## build

build code.
product code output to `build` directory.

```
npm run build
```

## clear

clear all cache files and build code.  
remove `dist`, `build`, `.cache` directory.

```
npm run clear
```
